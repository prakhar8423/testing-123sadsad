# Product

## Register
product

## Overview
This workspace builds a single-page web application (React + TypeScript + Tailwind,
Vite). The **full specification for the current app** — its features, screens, data
model, components, and design system — lives in **`PLAN.md`** at the workspace root.
`PLAN.md` is the source of truth for what to build; this file exists only to provide
design-context setup so the `impeccable` skill loads cleanly.

## Design direction
A modern, production-grade product UI. Follow the **DESIGN SYSTEM** section of
`PLAN.md` for the palette, typography, spacing, and layout. Compose from the shadcn
components in `app/src/components/ui`, use recharts for any data visualization, and
apply impeccable's color / typography / layout / motion rules for exceptional craft.

## Notes
- Do not stop to "initialize project context" — the context is already set up here.
- Never scaffold, install dependencies, or run a dev server; the platform owns the
  build and preview.
